# WireGuard           
TunSafe 下载: [传送门](https://tunsafe.com/download)          
### 一键安装 WireGuard        
Debian 9.x 一键安装 WireGuard        
```sh
wget https://raw.githubusercontent.com/koomox/devops/master/storage/linux/scripts/wireguard/debian9x_wireguard_v2.sh
chmod +x ./debian9x_wireguard_v2.sh
./debian9x_wireguard_v2.sh
```
Debian 9.x 自定义安装 WireGuard ，可添加多用户，添加防火墙规则。             
```sh
wget https://raw.githubusercontent.com/koomox/devops/master/storage/linux/scripts/wireguard/debian9x_wireguard_v1.sh
chmod +x ./debian9x_wireguard_v1.sh
./debian9x_wireguard_v1.sh
```
